## Facade 3D Lidar Modeling
Automatic segmentation of a building facade using 3D Lidar points in Matlab

## Features
- RANSAC Segmentation
- Contour Detection
- Lines Detection
- Points of Intersection

## Prerequisites
- Matlab : **2018a** or above
- Development Environment : Matlab App Designer
